# SpellCraft — UI kit

Recreation of the SpellCraft dictation/spelling app. Archetype: **kid**. Theme: **spellcraft** (dark-first).

## Screens covered
- Phone frame, dark gradient bg
- Practice screen: big word card with mic, pulse animation, replay/skip
- Grade picker (1–6)
- Daily stats
- Category list

## Notes
- Built from store screenshots + description. 6 grade levels, 10 themed categories, light/dark — this kit shows dark (the canonical brand state).
