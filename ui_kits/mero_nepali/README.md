# Mero Nepali — UI kit

Recreation of the Mero Nepali children's Nepali-learning app. Archetype: **kid**. Theme: **meronepali**.

## Screens covered
- Phone frame with status bar & bottom navigation
- Home: today's letter hero, module tiles (Alphabet / Numbers / Words / Comprehension), daily-game scene

## Notes
- Built from Play Store screenshots. Real Flutter widget tree not available at build time.
- Balloon + grass scene recreates the canonical "balloons carrying Devanagari letters" visual seen in the store listing.
